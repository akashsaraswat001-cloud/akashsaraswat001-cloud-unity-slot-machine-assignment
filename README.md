# Unity Slot Machine Assignment

## Overview
Unity-based slot machine game developed for the Underpin Technology Unity Developer Intern assignment.

## Features
- Randomized slot outcomes using RNG
- Matching-symbol win logic
- Credits and payout system
- Jackpot payout
- Smooth reel animations
- Lever interaction animation
- Sound effects
- Win celebration effects

## Project Structure
Assets/Scripts
Assets/Prefabs
Assets/Animations
Assets/UI
Assets/Sounds
Build/WebGL

## WebGL Build
WebGL build included in Build/WebGL folder.

## Approach
Developed using modular scripts:
- SlotManager
- ArrowBlink
- LightBlink

Provided assignment assets were used for symbols and interface.
